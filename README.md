# Asteroide em Pygame

Um jogo de asteroide simples desenvolvido em **Python + Pygame**, usado para demonstração sobre lógica de jogos, eventos, movimentação, colisões e loop de jogo.

---

## Funcionalidades

- Controle de nave
- Colisão com asteroides
- Pontuação
- Contador de vida
---

## Pré-requisitos

- Python 3.8+
- Pygame

Instalação:

```bash
pip install pygame
```
Como rodar
```bash
python pong.py
```
